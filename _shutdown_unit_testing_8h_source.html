34/** This class provides methods to access the unit testing support

36
37 This class provides an implementation of Shutdown interface that
38 has the ability to count the number times the Shutdown interface
39 is/was called instead of exiting. It can also alter the application's
40 exit code.
41
42 Notes:
43 o The Shutdown implementation assumes/uses the C Library 'exit()'
44 method.
45 */
46class ShutdownUnitTesting
47{
48public:
49 /** This method zero's the exit counter and sets the shutdown behavior

51 interface starts in the 'counter mode' with the count set to
52 zero.
53 */
54 static void clearAndUseCounter() noexcept;
55
56 /** This method returns the current call count AND clears the
57 call counter.
58 */
59 static size_t getAndClearCounter() noexcept;
60
61
62public:
63 /** This method allows the application to exit - but always with

67 */
68 static void setExitCode( int new_exit_code ) noexcept;

70 /** This method restores the exit/shutdown behavior to its 'non-test'
