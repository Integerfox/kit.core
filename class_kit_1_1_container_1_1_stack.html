Detailed Description

template<class ITEM>
class Kit::Container::Stack< ITEM >

This template class implements a Stack that has a fixed depth and stores copies of the data items.

Template Args: ITEM:= Type of the data stored in the Stack

◆ peekTop() [1/2]

template<class ITEM >
ITEM * Kit::Container::Stack< ITEM >::peekTop ( )
inlinenoexcept

Returns a POINTER to the top item of the Stack.

The state of the stack is unchanged. If the stack is empty then the method returns a null pointer. The pointer is valid until the next push or pop operation.

NOTE: The application CAN modify the item in place on the Stack. Use this power wisely!
