Kit::Container::KeyLiteralString Class Reference

This class provides a 'Key' wrapper for a C string literal. More...

Detailed Description

This class provides a 'Key' wrapper for a C string literal.

Keys of this type can used to compare against other KeyLiteralString, KeyStringBuffer, or Items that use a Kit::Text::String as their key.

#include <Key.h>
