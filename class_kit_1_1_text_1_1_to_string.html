Detailed Description

This static class is used to convert integer values to a null terminated string WITHOUT using snprintf() function calls.

This is for platforms/situations where it is not possible or undesirable (i.e. stack usage) to call snprintf.

Note: When using the template methods - you CAN NOT use a data type larger than KitTextToStringMaxUnsigned_T.

#include <ToString.h>

'padChar'The character to use to pad any unused leading characters in 'dstString'
Returns
A pointer to the beginning of the converted number when successful; else nullptr is returned on error (e.g. 'dstString' does not have sufficient space). The returned pointer is essentially a left justified string of the converted value. For a right justified result, use the original pointer passed as 'dstString'.

The template argument 'T' must be a signed integer type.
