## Welcome

Welcome to the Kit Embedded C++ Class Library documentation. Kit provides a comprehensive set of C++ classes and utilities specifically designed for embedded systems and resource-constrained environments.

## Key Features

- **Operating System Independent** OSAL layer with several RTOS/OS ports as well as bare metal applications
- **Hardware Independent** Drivers are decoupled from the underlying Microcontroller
- **Targeted to Microcontrollers** Designed specifically for resource-constrained environments such as micro-controllers
- **Off Target Automated Unit Tests** Over +80% of the code is unit tested as terminal application on Linux and Windows
- **Functional Simulator** No cost functional simulation of applications on Linux/Windows PCs.
