Detailed Description

template<class ITEM>
class Kit::Container::DList< ITEM >

This template class implements a singly linked list which maintains the ordering imposed on it by the application.

It is type-safe wrapper around the DListBase_ class.

NOTE: ITEM must be a subclass of Kit::Container::ExtendedItem.

#include <DList.h>
