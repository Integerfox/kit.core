Detailed Description

template<class KEYED_ITEM>
class Kit::Container::OrderedList< KEYED_ITEM >

This template class implements a double linked list which maintains the ordering imposed on it by the application.

It is type-safe wrapper around the OrderedListBase class.

NOTE: KEYED_ITEM must be a subclass of Kit::Container::KeyedItem.
