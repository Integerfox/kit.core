Detailed Description

template<int NUM_BYTES>
class Kit::Io::Ram::InputOutputAllocate< NUM_BYTES >

This concrete template convenience class extends the RAM InputOutput class to include allocating the Ring Buffer memory.

The template argument 'NUM_BYTES' is the number of 'useable' bytes.
