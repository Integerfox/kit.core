34/** This concrete class is a platform independent class that traverse a parent

43 */

56 /** This method allows the caller to walk the contents of the 'dirToList'
57 directory (assuming the entry is a directory). The default behavior is
58 to list only the current directory. The Caller can override the depth
59 of the traversal. Also, by default only the file name without its path
60 is returned. The caller can optionally have the traverse call only
61 return the name of files (omitting any directory names found) OR names
62 of directories (omitting any file names found). Returns true when
63 successfully and the entire traversal complete; else false is return if
64 their is an error (e.g. 'dirToList' is a file or does not exist) or the
65 'walker' aborted the traversal.
66
67 NOTE: The is no guaranteed order to the traversal, only that all items
68 per the specified depth will be traversed.
69
70 WARNING: On most platforms, the underlying OS calls dynamically
71 allocate memory. The OS clean-up after itself but there still
72 is the risk of failure do to lack of memory. Memory failures
73 are handled silently - but will not crash traversal.
74 */
