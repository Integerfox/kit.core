This 'convenience' method converts the a binary EDC or Hash value into an ASCII Hex string.

Returns true if the conversion is successful; else false is returned.

NOTE: If the IString is too small to hold the entire binary value as ASCII Hex the dstString argument is still updated with a truncated value and the method return false.
