Detailed Description

template<class SERVICE>
class Kit::Itc::SAP< SERVICE >

This concrete template class represents the interface to a ITC Service Access Point (SAP).

A SAP 'binds' a Request API with a Event Queue interface (aka a mailbox). In effect a Service is the 'fully-qualified' identifier for posting a message to a specific Request API.

#include <SAP.h>
