Detailed Description

This class provides an implementation for the CRC-32/MPEG-2 variant.

The CRC has following characteristics: o The polynomial is: x32 + x26 + x23 + x22 + x16 + x12 + x11 + x10 + x8 + x7 + x5 + x4 + x2 + x + 1 o The initial remainder is 0xFFFFFFFF. o The Data bytes are NOT reflected o The remainder is NOT reflected. o The final remainder is NOT XOR'd
