186 2) If 'lockRequest' is eLOCK, the method only returns if the MP is in
187 the unlocked state. In addition, when true is returned the MP is
188 put into the locked state.
189 3) If 'lockRequest' is eUNLOCK, the method always returns true and
190 the MP is left in the unlocked state.

192 This method is NOT thread safe.
