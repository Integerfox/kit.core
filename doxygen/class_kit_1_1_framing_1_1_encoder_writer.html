Detailed Description

This concrete class implements the basic framing/output logic for an Encoder.

The provided IDestination instances determines the "output destination"

When 'skipSendingSof' is set to true, the start-of-frame character will NOT be sent when startFrame() is called. This should ONLY be set to true when the framing protocol supports multiple SOF characters. See LineDecoder as example of such a protocol.
