21/** This concreate class implements a type safe MP for the data structure: Foo_T
22
23 The toJSON() format is:
24 ```
25 { name:"<mpname>", type:"<mptypestring>", valid:true|false, seqnum:nnnn, locked:true|false,
26 val: {
27 "upper":nnnn, "lower":nnnn, "result":nnnn, "isValid":true|false
28 }
29 }
30 ```
31
32 The "val" format for the fromJSON() format is:
33 ```
34 val: {"upper":nnnn, "lower":nnnn, "result":nnnn, "isValid":true|false }
35 ```
36
37 NOTES:
38 - The parent class (Kit::Dm::Mp::Scalar) provides the implementation
39 for all of the standard MP operations (e.g. read, write, attachObserver,
40 detachObserver, etc).
41
42 - The MpFoo class must provide the toJSON() and fromJSON() methods for the
43 Foo_T structure
44
45 - The MpFoo class provides additional "write" operations to enforce its
46 intended transaction paradigm.
47
48 - The toJSON() method supports only updating the specified/included KVPs,
49 e.g. "val":{"upper":12}" - is a read/modify operation that only updates
50 the 'upperLimit' field of the MP's data structure.
51
52*/
