Detailed Description

This abstract class represents the action that is executed by ITC framework in response to a Service completing the operation requested by a ServerMessage.

The ReturnHandler is executed when a Service invokes the returnToSender() operation of the message. Implementations of this class are the responsibility of the client (e.g. the client is using synchronous or asynchronous semantics when issuing the request).
