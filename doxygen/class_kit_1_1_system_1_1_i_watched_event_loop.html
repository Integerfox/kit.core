Detailed Description

This abstract interface defines the API for watching/monitoring an EventLoop for watchdog purposes.

The interface provides methods to start/stop monitoring and to perform periodic watchdog checks.

NOTE: The classes method should ONLY be called/invoked using the above macros

#include <IWatchedEventLoop.h>
