enum class Kit::Type::TraverserStatus
strong

This class defines a set of status(s) that are used with the traverser design pattern.

The traverser pattern is an adaptation of a internal iterator.

Enumerator
eABORT 

Indicates the client wants to cancel/terminate the traversal.

eCONTINUE 

Indicates the client wants to continue the traversal.
