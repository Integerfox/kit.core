Detailed Description

This concrete class implements a Line Writer stream using a previously opened IOutput stream.

NOTE: The LineWriter class does NOT provide any multi-thread mechanisms and/or protections. In addition, using an instance of the AtomicOutputApi interface as the underlying IOutput stream will NOT make output of the println() calls atomic! The intended way to have an 'Atomic' LineWriter is to use the AtomicOutput callback mechanism (i.e. requestOutputs()) and within the callback create the desired LineWriter using the supplied IOutput stream.
