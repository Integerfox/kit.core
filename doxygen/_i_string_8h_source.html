23/** This abstract class defines the operations that can be before on
24 a NULL terminated string. The intent is to provide a simple mechanism
25 for manipulating and "passing-around" strings rather than using error
26 prone explicit string operations.
27
28 NOTES:
29
30 o The memory management for the string data is determined by
31 the concrete sub-class!
32 o The 'IString' class IS A Kit::Container::Key, but it is NOT a
33 Kit::Container::xxxItem instance. Use the 'StringItem' class
34 for storing a string directly in a container.
35 */
