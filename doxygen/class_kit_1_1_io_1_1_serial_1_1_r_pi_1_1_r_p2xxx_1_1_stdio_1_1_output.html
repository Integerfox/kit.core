Detailed Description

This concrete class implements a Output stream using Raspberry Pi RP2xxx's SDK.

Multiple instances of this class can be created because there is only actual stdout stream.

Notes:

  1. The implement does NOT support the blocking semantics of the Kit::Io streams. The design decision was for the implementation to be compatible/useful on bare-metal systems, i.e. no threads required.
  2. The Application is RESPONSIBLE for calling stdio_init_all() on start-up of the application BEFORE any calls to this class (other than the constructor).
