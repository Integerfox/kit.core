Detailed Description

template<class LIST_ITEM>
class Kit::Container::SList< LIST_ITEM >

This template class implements a singly linked list which maintains the ordering imposed on it by the application.

It is type-safe wrapper around the SListBase class.

NOTE: LIST_ITEM must be a subclass of Kit::Container::ListItem.
