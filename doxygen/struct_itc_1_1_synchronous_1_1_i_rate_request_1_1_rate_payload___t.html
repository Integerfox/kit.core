uint32_t Itc::Synchronous::IRateRequest::RatePayload_T::flashRateMs

IN: The flash rate in milliseconds.

For 1Hz flash rate, this value would be 500ms (i.e. 500ms on, 500ms off). The valid flash rate range is [50ms, 5000ms]. Values outside of this range will cause the request to fail.
