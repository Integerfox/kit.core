Detailed Description

This class implements a brute force approach to initializing a 'pico semaphore' when the Kit C++ library is initialized.

This is to support statically allocated/created Kit::System::Semaphore instances. The issue here is that 'pico semaphore' need to be initialized AFTER main() is called.
