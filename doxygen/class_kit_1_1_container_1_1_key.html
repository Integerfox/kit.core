virtual int Kit::Container::Key::compareKey ( const Key & key) const
pure virtualnoexcept

Key Compare function.

Returns <0, 0, >0 if this key is less than, equal, or greater than respectively to the specified key! The actual type of the 'key' is up to the client sub-class that implements this interface. It is the responsibility of the sub-class to correctly define/interpret the data type of the key.

CAUTION: If the key being compared is NOT a compatible type or is not a valid key, then INT_MIN is returned. The caller of the method is responsible for checking for INT_MIN

Implemented in Kit::Container::KeyPlainType< DATATYPE >, and Kit::Container::KeyStringBuffer.
