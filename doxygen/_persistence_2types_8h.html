#define OPTION_KIT_PERSISTENCE_MEDIA_LITTLE_ENDIAN   1

Endianess for meta data that is persistently stored.

The default is Little Endian. To change to Big endian set the macro to 0.

NOTE: This ONLY applies to the meta-data - NOT the application data payload. The application is responsible for managing the Endianess of its data.
