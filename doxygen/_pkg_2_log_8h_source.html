11/** @file
12
13This file declares the Logging functions available to the KIT Logging Domain.
14
15CLIENTS SHOULD NOT call the Kit::Logging::Pkg::logfXXX() method directly. Instead
16the should use the KIT_LOGGING_LOG_XXX() macros. This allows the application
17to compile out the KIT library's usage of the Logging framework if desired.
18
19The compile switch do enable the Logging framework usage is:
20 USE_KIT_LOGGING_PKG_LOG_API
21
22When this switch is NOT defined the KIT_LOGGING_LOG_XXX() macros convert the logging
23calls into KIT_SYSTEM_TRACE_RESTRICTED_MSG calls
24
25NOTE: When the Application enables the Logging engine by defining
26 USE_KIT_LOGGING_PKG_LOG_API switch - it is required to provide the
27 "logging configuration", i.e. the assignment absolute values for
28 classification and package IDs.
