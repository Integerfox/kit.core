Detailed Description

This 'public' concrete class provides BIG ENDIAN byte ordering semantics for a Cursor.

The Cursor class properly handles multi-byte alignment and Endianess issues when copying multi-byte data to/from a memory buffer. See the CursorBase class for additional details
