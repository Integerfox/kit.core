Detailed Description

This abstract class represents a item that can be contained in an Map (aka a sorted list implemented using an AVL tree).

The client sub-class is required to implement the 'getKey()' method (from the parent class KeyedItem).

NOTE: Map Items CAN be used with SList or DList containers

#include <MapItem.h>
