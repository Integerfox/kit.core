Detailed Description

This class implements a brute force approach to initializing a 'pico mutex' when the Kit C++ library is initialized.

This is to support statically allocated/created Kit::System::Mutex instances. The issue here is that 'pico mutex' need to be initialized AFTER main() is called. The Pico-SDK solution for statically allocating mutex instances is C solution that does not work inside the Kit::System::Mutex object.
