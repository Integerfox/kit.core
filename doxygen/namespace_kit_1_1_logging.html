Detailed Description

The Logging namespace contains a logging framework as well as the Kit namespace specific logging identifiers and methods.

See The Kit::Logging::Framework namespace for details of the KIT library's approach to logging and its design.

See The Kit::Logging::Pkg namespace for details of the Kit namespace specific log identifiers and methods.
