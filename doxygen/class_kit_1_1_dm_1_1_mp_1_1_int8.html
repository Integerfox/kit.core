Detailed Description

This class provides a concrete implementation for a Point who's data is a int8_t.

The toJSON() format is:

{ name:"<mpname>", type:"<mptypestring>", valid:true|false, seqnum:nnnn, locked:true|false,
val:{ "dec":<numvalue>,
"hex":<hexvalue>
}
}

The "val" format for the fromJSON() format is:

val:<numvalue> OR val:"0x<hexvalue>"

NOTE: All methods in this class ARE thread Safe unless explicitly documented otherwise.
