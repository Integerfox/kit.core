Detailed Description

This concrete class provides a 'null' implementation of the IMedia interface.

The implementation is a 'happy path' implementation that does nothing, but does check/trap some basic errors. This class can be used to 'satisfy' persistent storage clients when there is no physical persistent storage available on the platform.
