Detailed Description

This concrete class implements the basic scanning/reading logic for a Decoder.

The provided ISource instance determines the "input source".

Constructor.

The size of the workBuffer determines how big of 'chunks' data is read from the "input source", i.e. it is a working buffer and does NOT have to be the size of the maximum possible input frame.
