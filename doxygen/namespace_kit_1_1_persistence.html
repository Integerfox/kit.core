Detailed Description

The 'Persistence' namespace is top level namespace for services, widgets, utilities, etc. that "application" layer (i.e. above the driver layer) persistent storage mechanisms.

Typedef Documentation

◆ Size_T

using Kit::Persistence::Size_T = uint32_t

This is the type for all size, lengths, offsets, etc.

for the Persistence namespace. NOTE: Is important that this type be a 'uintXX_t' type (must be an unsigned integer type). This is because other types like - size_t, unsigned, etc. - have different number of bits depending on the MCU/OS platform, Having target dependent sizes (e.g. think HW vs simulator) causes problems when the type (e.g. sizeof(size_t)) is used for the length of fields that get persistently stored.
