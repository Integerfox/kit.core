Detailed Description

This static class provides a collection of functions for operating on POSIX file descriptors.

NOTE: All file name/paths MUST be in native format (i.e. use '/' as the directory separator)

static bool Kit::Io::File::Posix::Fdio::readDirectory ( KitIoFileDirectory_T & hdl,
NameString & dstEntryName )
staticnoexcept

Reads 'next' entry in the directory.

The entry name is copied into 'dstEntryName'. If there are no more entries in the directory, then 'dstEntryName' is set to an empty string. The 'maxEntryNameSize' argument specifies the size - which needs to include space for the null terminator - of the 'dstEntryName' buffer.

Returns true if successful; else false if a file system error was encountered.
