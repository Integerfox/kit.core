Detailed Description

This file provides the Application level Logging Identifiers mappings for the contained Package(s).

This file needs to included by the "application's" kit_config.h file.

NOTE: Do not include OTHER HEADER FILES in this file! It will create circular dependencies issues. Unfortunately, this means that contents of this file must be 'hand-crafted'. The good news is that after the initial creation - the contents rarely change
