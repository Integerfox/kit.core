Detailed Description

Data structure used to exchange data between a Client and Server for the 'Foo' transaction.

The following are needed because the structure is used as a 'data source'

for a Model Point:

  • The comparison operation
  • The copy constructor and copy operator are implemented to ensure that any 'pad' bytes in the structure get zero'd out. This is desired because the default 'comparison operation' in the DM framework uses memcmp. A more sophisticated comparison approach can be used - but it requires the concrete Model Point to supply/implement the comparison method.
