28/** This concrete class implements the IChunk interface by storing two copies
29 of the Record's data, where each copy is CRC'd. This ensures that if power
30 fails during a write operation to persistent media - there will also be a
31 good 'previous' copy of the data available.
32
33 - The strength and bit width of the CRC is determined by the IEdc instance
34 provided in the class's constructor.
35
36 - The class requires two IMedia instances, one for each copy of the Record.
37 */
