21/** This protected base class implements a NON-TYPE-SAFE an ordered doubly
22 linked list. The order is determined by the compare of function of the
23 'Keyed Item' being stored in the list. It is intended to be the base class
24 for a type-safe template class. Having a non-templatized base class reduces
25 the memory footprint when an application has more than one type specific
26 OrderedListBase class.
27
28 NOTE: The insert algorithm is STUPID. It is a brute force walk of the existing
29 items in the list and inserting new item in order. If performance with
30 respect to ordering is an issue, you will need to use the Map container
31 which is implemented as a binary tree.
32 */

35protected:
