44 /** This method looks-up the model point instance by name and returns a

81
82 The method optional returns - via 'retMp' - a pointer to the Model Point
83 identified by 'mpname'. If the method false return, then 'retMp' has

96
97
98 The general input format:
99 \code
100

102 { name="<mpname>", valid:false } // Invalidates the MP
103 { name="<mpname>", valid:false, locked=true } // Invalidates the MP and locks the MP
104 { name="<mpname>", val:<value> } // Writes a new (valid) value to the MP
105 { name="<mpname>", val:<value>, locked=true } // Writes a new (valid) value to the MP and locks the MP

107
108 \endcode
