24/** This mostly concrete class provides the base implementation for a Point
25 who's data is a null terminated string. The concrete child class is
26 responsible for providing the string storage
27
28 The toJSON() format is:
29 \code
30
31 { name:"<mpname>", type:"<mptypestring>", valid:true|false, seqnum:nnnn, locked:true|false,
32 val:"<stringvalue>"
33 }
34
35 \endcode
36
37 The "val" format for the fromJSON() format is:
38 \code
39
40 val:"<stringvalue>"
41
42 \endcode
43 NOTE: All methods in this class ARE thread Safe unless explicitly
44 documented otherwise.
45
46 NOTE: The MP's null terminator for the string storage IS imported/exported.
47 */
