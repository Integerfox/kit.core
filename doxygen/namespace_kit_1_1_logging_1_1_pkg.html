Detailed Description

The Pkg namespace provides the KIT library's "package" specific logging identifiers and methods. In this context - package - means the Kit namespace.

See The Kit::Logging::Framework namespace for details of the KIT library's approach to logging and its design.

This enumeration defines the logging Classification IDs for the KIT Logging Package.

NOTE: The valid range for a Classification ID is [1-32].

Note: Each symbol's length must be less than or equal to 16 characters 123456789 123456

Parameters
ClassificationIdEnum
FATALA fatal unrecoverable error occurred. The system will be 'shutdown'. What 'shutdown' means is platform dependent.
WARNINGA recoverable error and/or an unexpected condition occurred
EVENTA significant event occurred.
INFOInformational. Typically used for detailed troubleshooting and is not recommended to be enabled by default
