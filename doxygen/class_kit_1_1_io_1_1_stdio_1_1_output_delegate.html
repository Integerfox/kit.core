Detailed Description

This concrete class implements a Output stream using the underlying platform's native OS interface' sfor the C library's stdout and stderr streams.

Design Note: It is important that this class be concrete class that can be created by application code that is target/platform independent. This is why the data type for 'm_outFd' member variable is defined using the LHeader pattern.
