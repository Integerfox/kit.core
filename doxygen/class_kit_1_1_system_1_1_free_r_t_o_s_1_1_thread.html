◆ makeNativeMainThreadAKitThread()

static void Kit::System::FreeRTOS::Thread::makeNativeMainThreadAKitThread ( )
staticnoexcept

This is helper method to 'convert' the first/main FreeRTOS thread to a KIT thread.

The method can be called many times - but it does the 'conversation' once. The motivation for this method was working with the Arduino platform/framework where it creates the first/main FreeRTOS thread.
