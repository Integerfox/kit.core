24/** This class provides a concrete implementation for a Point who's data is 32
25 bit unsigned reference counter. A reference counter can be increment or
26 decremented. When incremented, the value is clamped at 2^32 -1 (not allowed
27 to overflow). When decremented, the value is clamped at zero (not allowed
28 to underflow). In addition, change notifications are only generated on the
29 following transitions:
30
31 1) Invalid state to the Valid state
32 2) Valid state to the Invalid state
33 3) Transition to zero
34 4) Transition from zero to value greater than zero
35
36 The toJSON() format is:
37 \code
38
39 { name:"<mpname>", type:"<mptypestring>", valid:true|false, seqnum:nnnn, locked:true|false,
40 val:<numvalue>
41 }
42
43 \endcode
44
45 The "val" format for the fromJSON() format is:
46 \code
47
48 val:"[<act>]<numvalue>
49 where <act> can be:
50 "+" -->increment the counter
51 "-" -->decrement the counter
52 (no prefix) -->reset the counter to the specified value
53
54 Examples:
55 { name:"mp_visitors", val:"+2" } // Increments the point by 2
56 { name:"mp_visitors", val:"-1" } // Decrements the point by 1
57 { name:"mp_visitors", val:0 } // Resets the counter to zero
58
59 \endcode
60
61 NOTE: All methods in this class ARE thread Safe unless explicitly
62 documented otherwise.
63 */
