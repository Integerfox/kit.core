Detailed Description

template<class DATATYPE>
class Kit::Container::KeyPlainType< DATATYPE >

This template class is used to generate Key classes for most of the C/C++ primitive data types.

This class is NOT intended for general use - it is used create the typedefs that follow.

#include <Key.h>
