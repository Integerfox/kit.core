Detailed Description

This concrete class provides a 'null' implementation of the NV::IApi interface.

The implementation reports success for all operations while not storing any written data. Reads return zero-filled data when the destination buffer is large enough. This allows the driver to be used as a passive 'stub' when a platform-specific driver is not available.
