Detailed Description

This concrete class implements the IChunk interface by storing two copies of the Record's data, where each copy is CRC'd.

This ensures that if power fails during a write operation to persistent media - there will also be a good 'previous' copy of the data available.

  • The strength and bit width of the CRC is determined by the IEdc instance provided in the class's constructor.
  • The class requires two IMedia instances, one for each copy of the Record.
