Detailed Description

This abstract class defines the public interface for a Record instance.

A Record is the atomic-unit that is managed in Persistent storage media.

◆ start()

virtual bool Kit::Persistence::Record::IRecord::start ( Kit::EventQueue::IQueue & myEventQueue)
pure virtualnoexcept

This method is used to start/initialize the Record.

It is typically only called once at the startup of the application. However, start() can be called after a previous call to the stop() method.

This method is called when the Record Server is 'opened'.

The 'myEventQueue' is a reference to the Record Server's Event Queue, i.e. the event loop for the thread that the Record Server executes in. Example usage of 'myEventQueue' is if the IRecord needs/uses a software timer

The method returns true on success; else false is returned when an error occurred during the start.

Implemented in Kit::Dm::Persistence::Record, Kit::Persistence::Record::Journal::EntryRecord, and Kit::Persistence::Record::Journal::HeadRecord.
