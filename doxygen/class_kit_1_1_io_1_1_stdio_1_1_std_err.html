This concrete class represents the C-library 'stderr' stream. More...

Detailed Description

This concrete class represents the C-library 'stderr' stream.

NOTE: Many instances of this class may be created - even though there is only ONE stderr. This is OK because all instance reference the systems/platform 'stderr'

#include <StdErr.h>
