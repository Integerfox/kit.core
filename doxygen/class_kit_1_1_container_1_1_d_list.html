Detailed Description

template<class EXT_ITEM>
class Kit::Container::DList< EXT_ITEM >

This template class implements a double linked list which maintains the ordering imposed on it by the application.

It is type-safe wrapper around the DListBase class.

NOTE: EXT_ITEM must be a subclass of Kit::Container::ExtendedItem.
