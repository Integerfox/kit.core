Detailed Description

This concrete class implements the IRecord interface that manages a collection of Entries that are timestamp and oldest entries are silently lost when the IMedia storage is 'full'.

This class is responsible for managing the 'head pointer' for the logical RingBuffer that is used to manage multiple entries.
