Detailed Description

The 'Persistence' namespace provides a persistent storage mechanism for Model Points. The KIT Persistence framework is used for base persistent storage.

A KIT Persistence Record is used to store 1 or more model points to persistent storage. With respect to persistence storage - a Record is an "atomic unit"
