# Projects.Examples.Container {#projects_examples_container}

\brief Root directory for the KIT Container examples.

KIT library supports numerous container type such as:

- Limitless Singly and Doubly linked lists without dynamic memory
- Limitless AVL Binary Tree (aka a 'map') without dynamic memory
- Ring Buffers with a fix size.  Several variants: single threaded, multi-threaded, and ISR safe.
- Stack (with a fix depth)

The contained directories are for [Container](https://github.com/Integerfox/kit.core/blob/main/src/Kit/Container/README.md) examples.
