# Kit::TShell::Command
@brief namespace description for Kit::TShell::Command
@namespace Kit::TShell::Command @brief

The Command namespace provides a collection of concrete TShell commands. The
application is free to choose which commands to include in its TShell.

NOTE: This namespace is **not** the only location/collection of TShell commands.
      Individual sub-system, widgets, etc. can have their own feature specific
      command(s).
