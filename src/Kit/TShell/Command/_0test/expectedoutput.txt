bye [<exitcode>]
bye tshell
  Requests to exit the application. The default <exitcode> is '0'. Using the
  the 'tshell' argument only exits the TShell.  NOTE: The <exitcode> and
  what happens when the only the TShell is exited is application specific
 
echo ["<text>"]
echo - ["<text>"]
  Outputs the optionally specified text with the current elapsed time
  prepended to the text.  The '-' option is used to suppress the elapsed time
  output (this is helpful when constructing unit tests)
 
help [* <cmd>]
  Displays list of all supported commands and optionally their detailed help. If
  the second argument is a command, then the detailed help for that command will
  be displayed.
 
trace [on|off]
trace section (on|off) <sect1> [<sect2>]...
trace threadfilters [<threadname1> [<threadname2>]]...
trace level (eNONE|eBRIEF|eINFO|eVERBOSE|eMAX)
trace here|revert
  Enables/Disables the Kit::System::Trace engine and manages the section',
  information level, and thread filter options.  See the Kit::System::Trace
  interface for details on how the trace engine works.
 
wait <delay>
  Waits for 'delay' milliseconds before returning.  This is useful when issuing
  a batch of console commands and you need a delay between the execution of 
  commands.
